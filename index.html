Cualquier teléfono con navegador
puede conectar con la página web
y hacer uso del sistema de control.
Los lenguajes web prestan diversas formas
de gestionar los datos de los sensores,
ya sea tanto usando bases de datos
ó puramente PHP y bloc de notas,
como scripts en JS y Phyton.
Ahora la computadora recibe los datos
de la programadora estableciendo
una comunicación COM ó serie
por el puerto de red RJ45.
El controlador ethernet ENC28j60
negocia el intercambio de datos con la PC
ofreciendo su MAC para establecer una IP.
El PiC24F es el cerebro del proyecto,
ya que se encarga tanto de leer cambios
en las E/S de los sensores/actuadores,
como de enviarlos hacia el ENC28j60
por el TCP/IP Stack.
Un módulo con un relé optoacoplado
escucha al PiC por un transceptor RF
para actuar sobre el switch de luz.
Un sensor de humedad equipado con RF
envía de manera constante lecturas
basadas en una diferencia de capacidad.
